# Java Spring OTEC Admin

Aplicación web desarrollada en **Java Spring Boot** para la gestión administrativa de una OTEC. Incluye módulos para administrar estudiantes, cursos y matrículas, con una arquitectura clara, mantenible y escalable.

 Tecnologías utilizadas
- Java 25
- Spring Boot
- Spring MVC
- Spring Data JPA
- Thymeleaf
- Bootstrap 5
- Maven
- MariaDB / MySQL
